Cluster Members

This dialog allows you to control the number of nodes in the cluster and their respective IP addresses.
Please make sure that the provided IP addresses correspond to the network addresses of the rings specified in the "Communication Layer" dialog.

Note: the number of nodes for HANA scale-up clusters is fixed and cannot be changed.

On passwordless SSH

When the "Next" button is clicked, the Wizard will check if it can connect to the remote node(s) via SSH protocol without a password. If it fails to do so, it will require the root password for the remote node, connect to it and propagate its own SSH keys (both private and public) to the slave node(s).
