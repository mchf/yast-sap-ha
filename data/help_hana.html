HANA Configuration

This page defines the parameters specific for SAP HANA installations.

Virtual IP Address is the IP address that the current SAP HANA primary instance has.

Prefer Site Takeover defines, if the secondary instance should take over, if the primary instance fails. If "false", prefer local restart of SAP HANA.

Automatic Registration defines, if the former primary should be registered as the new secondary after the takeover procedure.
